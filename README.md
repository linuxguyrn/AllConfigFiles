## Setup Commands:-
sudo dnf update
sudo dnf install starship
sudo dnf install kitty
sudo dnf install neovim git curl bash nodejs npm python3 go ripgrep lazygit bottom
git clone --depth 1 https://github.com/AstroNvim/template ~/.config/nvim

## Other Stuff:-
- Enable Wobbly Windows in KDE system settings.
- Download starship.toml and place it in ~/.config
- Download Nerd Font
- Install Astronvim on neovim
